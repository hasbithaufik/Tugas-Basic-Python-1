
Catatan sebelum penggunaan:
untuk menggunakan program ini, opsi "less secure apps access" di Gmail anda harus diubah menjadi "ON"
langkahnya:
	1. jalankan link berikut di browser anda: https://myaccount.google.com/security?gar=1
	2. scroll ke bawah, kemudian cari opsi "less secure apps access" 
	3. Klik, dan ubah "Off" menjadi "ON"
